# ✨ Ежедневная Искра — Daily Challenge Generator

Веб-приложение для генерации персонализированных ежедневных микрозадач на основе твоего текущего уровня энергии. Работает на базе искусственного интеллекта Google Gemini.

---

## 🚀 О проекте

**Ежедневная Искра** помогает начать день с правильного настроя. Просто выбери, как ты себя чувствуешь — и получи 3 небольших задачи, идеально подходящих под твоё состояние прямо сейчас.

---

## ✨ Возможности

- 🎚️ **Выбор уровня энергии** — три варианта: Низкая 😴 / Средняя 😊 / Высокая ⚡
- 🤖 **Генерация задач с помощью ИИ** — Google Gemini создаёт уникальные задачи под твой настрой
- ✅ **Отметка выполненных задач** — нажми на кнопку рядом с задачей, чтобы зачеркнуть её
- 📊 **Прогресс-бар** — отслеживай сколько задач уже выполнено (например 1/3)
- 🎉 **Конфетти** — праздничная анимация когда все задачи выполнены
- 🔄 **Кнопка "Начать заново"** — сбрось прогресс и выбери новый день
- 📱 **Адаптивный дизайн** — отлично работает на телефоне и компьютере

---

## 🛠️ Технологии

### Frontend
- **React** + **TypeScript**
- **Tailwind CSS** — стилизация с пастельной цветовой схемой
- **Framer Motion** — плавные анимации карточек
- **canvas-confetti** — анимация конфетти при завершении всех задач
- **TanStack Query** — управление состоянием запросов
- **Wouter** — маршрутизация

### Backend
- **Node.js** + **Express**
- **Google Gemini API** (`gemini-2.0-flash`) — генерация задач
- **Zod** — валидация данных
- **TypeScript** — типизация на обоих сторонах

## ⚙️ Установка и запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/beckkyway/daily-mood-tasks.git
cd daily-mood-tasks
```

### 2. Установи зависимости

```bash
npm install
```

### 3. Настрой переменные окружения

Создай файл `.env` и добавь API ключ Google:

```env
GOOGLE_API_KEY=твой_ключ_здесь
```

> Получить ключ можно на [Google AI Studio](https://aistudio.google.com/app/apikey)

### 4. Запусти приложение

```bash
npm run dev
```

Приложение откроется на `http://localhost:5000`

## 🔑 Как получить Google API Key

1. Зайди на [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Нажми **"Create API key"**
3. Скопируй ключ и добавь в переменные окружения
